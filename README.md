# Školský informačný systém (EduPage Clone)

Tento projekt je webová aplikácia postavená na frameworku **Flask** s využitím **PostgreSQL** databázy.

### Hlavné súčasti:
- **Backend**: Python (Flask)
- **Databáza**: PostgreSQL (schéma v `database_schema.sql`)
- **Frontend**: HTML5 + CSS3 (v priečinku `templates`)

### Inštalácia a spustenie:
1. Nainštalujte knižnice: `pip install -r requirements.txt`
2. Importujte databázu pomocou `database_schema.sql`.
3. Spustite aplikáciu príkazom `python app.py`.
